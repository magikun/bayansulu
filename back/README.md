# 🐍 Bayan Sulu Kids - Python FastAPI Backend

Этот каталог содержит серверный бэкенд для проекта **Bayan Sulu Kids**, написанный на высокопроизводительном фреймворке **Python FastAPI**. 

Бэкенд расширяет возможности MVP, добавляя синхронизацию локального прогресса, глобальную таблицу лидеров (Leaderboard) и систему генерации купонов Баян Сулу.

## 🚀 Быстрый запуск бэкенда

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python 3.9 или выше. Перейдите в каталог `back` и создайте виртуальное окружение:

```bash
cd back
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

### 2. Установка зависимостей
Установите все необходимые библиотеки, перечисленные в `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 3. Запуск сервера разработки
Запустите сервер с автоперезагрузкой с помощью `uvicorn`:

```bash
uvicorn main:app --reload --port 8000
```

После этого:
* API будет доступно по адресу: **http://localhost:8000**
* Интерактивная документация Swagger (где можно протестировать запросы): **http://localhost:8000/docs**
* Альтернативная документация ReDoc: **http://localhost:8000/redoc**

---

## 🗺️ Конечные точки (API Endpoints)

*   `GET /` — Проверка статуса сервера.
*   `GET /api/prizes` — Получение каталога доступных призов и скидок АО «Баян Сулу».
*   `POST /api/player/sync` — Синхронизация данных игрока (имя, уровень, монеты, достижения) между локальным хранилищем браузера и сервером.
*   `GET /api/leaderboard` — Запрос глобальной таблицы лидеров для повышения вовлеченности детей (соревновательный элемент).
*   `POST /api/coupons/purchase` — Обмен заработанных монет Botacoins на уникальный штрих-код/купон для покупки сладостей со скидкой.
